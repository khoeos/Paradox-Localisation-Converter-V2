Paradox localisation file converter

How does it work ?

You can learn more in the github readme accessible with the button in the top right corner.
But to explain it simply, this app generate language file from english to avoid translation tag when files for your language are missing.
Enter your mod folder location, and select which language you want. The app will take the English localisation files, adapt it to your desired language and try to add them in the folders. If the file already exists it will not replace it.

  • A .addedFiles file is added to each mod folder with the new Files
  • If the mod use synced_localisation or replace folder, the script will ignore the files inside
  • The script work for the entire mod folder but also for single mod.
  • For use with another paradox game, look into the about tab.

Folder where all your mods are located, or a single mod (modpack for example)
